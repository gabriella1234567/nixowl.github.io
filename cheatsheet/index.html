HTML

HTML is the standard markup language for creating Web pages. HTML stands for Hyper Text Markup Language. HTML describes the structure of a Web page. HTML consists of a series of elements that tell the browser how to display the content. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Creating a new HTML document

Tag Description
<!DOCTYPE html> Defines the document type. You can see this at the start of an html document, before the opening <html> tag.
<link rel="stylesheet" href="mystyle.css"> Links to an external stylesheet. The "rel" attribute specifies the relationship between the HTML document and the linked document. In this case, it's a stylesheet. The "href" attribute specifies the location of the linked document. In this case, it's a file called mystyle.css in the same folder as the HTML file.
<meta charset="utf-8"> Specifies the character encoding for the HTML document. In this case, "utf-8" is used, which is a widely supported character encoding that can represent most characters from all writing systems.
<meta name="keywords" content="keywords"> Meta "tags" for your site, used by search engines.
<meta name="description" content="description">
 This is the description for a site that you see in search results on Google.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
 Sets the viewport to make your website look good on all websites, where "width" is set to follow the width of the device and initial-scale is the default zoom level. Important for responsive design.

Creating a table

Tag Description
<table>, <th>, <tr>, <td> The whole table is wrapped in the <table> tag. You wrap each row in <tr>, and then populate the rows with either <th> which is table head, like the "tags" and "description" in this table, or <td> for table data, which is any cell that isn't the header. These tags are not self closing.
colspan and rowspan These can be used on <th> and <td> like this: <td colspan="2"> which would make that cell take up two columns, so two cells horizontally. <td rowspan="2"> would make it take up two rows, so two cells vertically.

Creating links

Tag Description
<a href="#id"> Links are made with the <a> tag. The "href" attribute specifies the URL of the page the link goes to. For an internal link, you use # and the ID you gave the element you're linking to.
<a href="url" target="_blank"> The "target" attribute specifies where to open the linked document (in this case, a new tab). The "_blank" value specifies that the linked document will open in a new window or tab. You can also use "_self" to open the linked document in the same frame as it was clicked on, or "_parent" to open the linked document in the parent frame, or "_top" to open the linked document in the full body of the window.

CSS

CSS is the language we use to style an HTML document. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work as it can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

I want to.. Tags Description
Make the browser accurately calculate element sizes box-sizing: border-box By default, the width and height of an element is calculated like this: width + padding + border = actual width of an element. This can be annoying when you're trying to make a responsive design. By setting box-sizing to border-box, the width and height of an element is calculated like this: width = actual width of an element. This means that the padding and border are included in the width and height.
Change how elements interact with the flow of the document position: static, relative, absolute, fixed Static is the default, and it means that the element will be positioned according to the normal flow of the page. Relative means that the element will be positioned relative to its normal position. Absolute means that the element will be positioned relative to its closest positioned ancestor. Fixed means that the element will be positioned relative to the browser window.
Change how elements align with each other display: block, inline-block, inline Block elements take up the full width available, and have a line break before and after them. Inline-block elements take up only as much width as they need, and don't have a line break before or after them. Inline elements take up only as much width as they need, and have a line break before and after them. You can change the width of block and inline-block elements, but not inline elements.
Use flexbox display: flex or inline-flex Flexbox is a layout mode that arranges elements in a predictable way for different screen sizes and orientations. It's great for responsive design. See full image here.

JavaScript

JavaScript is the programming language of HTML and the Web. JavaScript is easy to learn.

I want to.. Tags Description
Change the content of an HTML element document.getElementById(id).innerHTML = new HTML Use the document.getElementById(id) method to find the element you want to change. Then use the innerHTML property to change the content of that element.
Change the style of an HTML element document.getElementById(id).style.property = new style Use the same method as above, but instead of innerHTML, use style.property. For example, document.getElementById("demo").style.fontSize = "35px".
Change what happens on hover or click document.getElementById(id).onmouseover = function(){myScript}; Use the same method as above, but instead of innerHTML, use onmouseover or onclick. For example, document.getElementById("demo").onmouseover = function(){this.innerHTML = "Hello World";};
